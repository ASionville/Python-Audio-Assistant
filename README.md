# Python-Audio-Assistant
A web-based audio assistant in Python 3 

**Requirements :**
You _need_ to install all of these packages from pip :
- speech_recognition
- playsound
- gtts
- wolframalpha
- googlesearch
- webbrowser
- googletrans
- bs4

and you 'll need PyAudio from here :
https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio
